# Sales Forecasting & Demand Intelligence Dashboard

## Project Overview
This project analyzes historical sales data and predicts future sales using the SARIMA forecasting model. It also detects sales anomalies and identifies high-demand product segments through an interactive Streamlit dashboard.

## Features
- Sales Overview Dashboard
- Monthly Sales Trend
- Forecast Explorer
- SARIMA Sales Forecasting
- Sales Anomaly Detection
- Product Demand Segmentation
- Interactive Filters

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels (SARIMA)

## Project Files
- app.py
- requirements.txt
- train.csv
- Sales_Forecasting_Analysis.ipynb
- summary.docx
